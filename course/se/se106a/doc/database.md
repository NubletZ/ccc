# 資料庫

關聯式資料庫 v.s 文件資料庫

1. PostgresSQL 
2. MongoDB
3. Amazon DynamoDB

## PostgreSQL

1. 關聯式資料庫系統
2. 可以支援 JSON 物件 (9.2 版之後) 與速度更快的 JSONB 物件 (9.4 版之後, JSONB可以建立索引)
3. 支援全文檢索
